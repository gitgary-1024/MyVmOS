# 手写汇编器实现总结 - IR 直接编译为机器码

## 🎯 目标

不依赖外部汇编器（如 NASM），**手写 IR 到 x86-64 机器码的转换逻辑**，生成纯二进制对象文件，便于在 x86 VM 中运行。

#### 头文件声明
[`Assembler.h`](file://d:\ClE\debugOS\MyOS\Compiler\include\IR\Assembler.h)

```cpp
class Assembler {
private:
    std::vector<IRNode> irNodes;
    std::vector<uint8_t> machineCode;  // 生成的机器码
    std::unordered_map<std::string, uint64_t> labelToAddress;
    std::unordered_map<std::string, uint64_t> variableOffsets;
    
    // 寄存器编码枚举
    enum Reg64 {
        RAX = 0, RCX = 1, RDX = 2, RBX = 3,
        RSP = 4, RBP = 5, RSI = 6, RDI = 7
    };
    
public:
    Assembler(const std::vector<IRNode>& ir);
    std::vector<uint8_t> compile();
    bool saveToFile(const std::string& filename);
    void printMachineCode() const;
};
```

#### 核心方法

1. **`compile()`** - 两遍扫描编译
   - 第一遍：收集标签地址和变量偏移
   - 第二遍：编码所有 IR 指令为机器码

2. **`saveToFile()`** - 保存为二进制文件
3. **`printMachineCode()`** - 打印十六进制机器码（调试用）

bool File::compileToObjectFile(const std::string& outputFile) {

        return true;

        return false;
    }

## 🔬 技术实现细节

### 1. **x86-64 指令编码规则**

#### REX 前缀

64 位模式需要 REX 前缀来访问 64 位寄存器：

```
REX.W = 0x48  // 启用 64 位操作数宽度
```

#### MOV 指令编码示例

```cpp
void Assembler::encodeMOV(const IRNode& node) {
    std::string dest = node.operands[0];
    std::string src = node.operands[1];
    
    // 情况 1: MOV reg, imm (寄存器 = 立即数)
    if (isRegister(dest) && isNumber(src)) {
        int64_t imm = parseNumber(src);
        int reg = getRegCode(dest);
        
        // REX.W + B8+rd imm64
        emitByte(0x48);      // REX.W
        emitByte(0xB8 + reg);
        emitQWord(imm);      // 64 位立即数
    }
    
    // 情况 2: MOV reg, reg (寄存器 = 寄存器)
    else if (isRegister(dest) && isRegister(src)) {
        int destReg = getRegCode(dest);
        int srcReg = getRegCode(src);
        
        // REX.W + 89 C0+src*8+dest
        emitByte(0x48);
        emitByte(0x89);
        emitByte(0xC0 + (srcReg << 3) + destReg);
    }
}
```

### 2. **小端序编码**

```cpp
void Assembler::emitDWord(uint32_t dword) {
    // 小端序：低位在前
    emitByte((dword >> 0) & 0xFF);
    emitByte((dword >> 8) & 0xFF);
    emitByte((dword >> 16) & 0xFF);
    emitByte((dword >> 24) & 0xFF);
}

void Assembler::emitQWord(uint64_t qword) {
    for (int i = 0; i < 8; i++) {
        emitByte((qword >> (i * 8)) & 0xFF);
    }
}
```

### 3. **相对跳转地址计算**

```cpp
void Assembler::encodeJMP(const IRNode& node) {
    std::string label = node.operands[0];
    auto it = labelToAddress.find(label);
    
    if (it != labelToAddress.end()) {
        // 相对跳转：E9 rel32
        int64_t targetAddr = it->second;
        int64_t currentPos = currentAddress + 5; // JMP 指令长度
        int32_t relOffset = targetAddr - currentPos;
        
        emitByte(0xE9);
        emitDWord(relOffset);
    }
}
```

### 4. **寄存器映射**

```cpp
int Assembler::getRegCode(const std::string& reg) {
    std::string r = reg;
    std::transform(r.begin(), r.end(), r.begin(), ::tolower);
    
    if (r == "rax" || r == "eax" || r == "ax" || r == "al") return RAX;  // 0
    if (r == "rcx" || r == "ecx" || r == "cx" || r == "cl") return RCX;  // 1
    if (r == "rdx" || r == "edx" || r == "dx" || r == "dl") return RDX;  // 2
    if (r == "rbx" || r == "ebx" || r == "bx" || r == "bl") return RBX;  // 3
    if (r == "rsp" || r == "esp" || r == "sp") return RSP;               // 4
    if (r == "rbp" || r == "ebp" || r == "bp") return RBP;               // 5
    if (r == "rsi" || r == "esi" || r == "si") return RSI;               // 6
    if (r == "rdi" || r == "edi" || r == "di") return RDI;               // 7
    
    return RAX; // 默认
}
```

## ⚠️ 当前限制

### 1. **内存访问简化**

当前对全局变量的访问使用占位符地址：

```cpp
// MOV [mem], reg
emitByte(0x48);
emitByte(0x89);
emitByte(0x05);
emitDWord(0);  // 占位符，需要后续填充
```

**改进方向：**
- 使用 RIP 相对寻址
- 或在 VM 中建立符号表进行重定位

### 2. **部分指令未完全实现**

- `IDIV`：使用 NOP 占位
- `CALL`：相对地址占位
- `LEA`：地址占位

### 3. **没有重定位信息**

生成的代码假设从地址 0 开始执行，需要 VM 负责加载到正确位置。
