## 🔧 技术细节

### 寄存器映射机制

```cpp
// File::compileToAssembly 中
std::unordered_map<std::string, std::string> regMap;
regMap["eax"] = "rax";
regMap["ebx"] = "rbx";
regMap["ecx"] = "rcx";
regMap["edx"] = "rdx";
regMap["esi"] = "rsi";
regMap["edi"] = "rdi";
regMap["ebp"] = "rbp";
regMap["esp"] = "rsp";

IRToAssembly assembler(IRnodes, regMap, useIntelSyntax, withComments);
```

### getPhysicalReg 函数

```cpp
std::string IRToAssembly::getPhysicalReg(const std::string& vreg) {
    // 如果寄存器映射中存在，使用映射的寄存器
    auto it = regMap.find(vreg);
    if (it != regMap.end()) {
        return it->second;
    }
    
    // 如果没有映射，假设是变量名，返回内存地址
    return "[" + vreg + "]";
}
```

### compileToBinary 实现流程

```cpp
bool File::compileToBinary(const std::string& outputFile, bool useIntelSyntax) {
    // 1. 生成汇编代码
    std::string assembly = compileToAssembly(useIntelSyntax, false);
    
    // 2. 添加段声明和数据定义
    std::stringstream fullAsm;
    fullAsm << "section .data\n";
    fullAsm << "    x: dq 0\n";
    fullAsm << "    y: dq 0\n";
    fullAsm << "    result: dq 0\n";
    fullAsm << "\n";
    fullAsm << assembly;
    
    // 3. 写入临时汇编文件
    std::ofstream asmOut("temp_output.asm");
    asmOut << fullAsm.str();
    asmOut.close();
    
    // 4. 调用 NASM 汇编器
    system("nasm -f win64 temp_output.asm -o temp_output.o");
    
    // 5. 调用链接器
    system("g++ -o output.exe temp_output.o");
    
    // 6. 清理临时文件
    std::remove("temp_output.asm");
    std::remove("temp_output.o");
}
```

## ⚠️ 已知限制

### 1. **64 位模式下的内存访问**

**问题：** 在 x86-64 模式下，访问全局变量需要使用 RIP 相对寻址，但当前生成的是绝对地址访问。

**错误示例：**
```asm
mov [x], rax    ; ❌ 在 64 位模式下会产生重定位错误
```

**需要改为：**
```asm
lea rax, [rel x]    ; ✓ RIP 相对寻址
mov [rax], rbx
```

### 2. **解决方案选项**

#### 方案 A：修改 IR 生成器（推荐）
- 让 IR 生成器使用局部变量（栈分配）而不是全局变量
- 修改 `VariableDeclaration` 的 IR 生成逻辑
- 使用 `[rbp-offset]` 形式访问局部变量

#### 方案 B：修改汇编生成器
- 识别全局变量访问
- 自动生成 RIP 相对寻址代码
- 需要更复杂的地址计算

#### 方案 C：使用 32 位模式
- 生成 32 位汇编代码
- 使用 `-f win32` 格式编译
- 链接时使用相应选项
