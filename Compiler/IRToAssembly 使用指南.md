# IRToAssembly 使用指南

## 📋 概述

`IRToAssembly` 是一个将中间表示（IR）转换为 x86-64 汇编代码的代码生成器。它支持：

- ✅ **Intel 语法**和**AT&T 语法**
- ✅ **所有基础 IR 指令**（MOV、算术运算、逻辑运算、跳转等）
- ✅ **寄存器映射**（虚拟寄存器 → 物理寄存器）
- ✅ **内存操作数**支持
- ✅ **可选的注释生成**

---

## 🔧 使用方法

### 基本示例

```cpp
#include "IR/IRbase.h"
#include "IR/IRToAssembly.h"
#include <vector>
#include <iostream>

int main() {
    // 1. 创建 IR 指令列表
    std::vector<IRNode> irNodes;
    
    // 添加指令：eax = 5 + 3
    irNodes.push_back(IRNode(IROp::MOV, {"eax", "5"}, "加载 5"));
    irNodes.push_back(IRNode(IROp::MOV, {"ebx", "3"}, "加载 3"));
    irNodes.push_back(IRNode(IROp::ADD, {"eax", "ebx"}, "相加"));
    
    // 2. 创建汇编生成器
    IRToAssembly assembler(irNodes);
    
    // 3. 编译为汇编代码
    std::string assembly = assembler.compile();
    
    // 4. 输出结果
    std::cout << assembly << std::endl;
    
    return 0;
}
```

### 输出示例（Intel 语法）

```asm
; Generated by IRToAssembly Compiler
; x86-64 Assembly Code
; Syntax: Intel

section .text
global main

    mov eax, 5
    ; IR: mov eax, 5
    mov ebx, 3
    ; IR: mov ebx, 3
    add eax, ebx
    ; IR: add eax, ebx
```

## ⚙️ 配置选项

### 构造函数参数

```cpp
IRToAssembly(
    const std::vector<IRNode>& ir,                    // IR 指令列表
    const std::unordered_map<std::string, std::string>& regMapping = {}, // 寄存器映射
    bool intelSyntax = true,                          // true=Intel 语法，false=AT&T 语法
    bool withComments = true                          // 是否生成注释
);
```

### 示例：使用 AT&T 语法

```cpp
// 使用 AT&T 语法
IRToAssembly assembler(irNodes, {}, false, true);
std::string assembly = assembler.compile();
```

**输出：**
```asm
    mov $5, %eax
    # IR: mov eax, 5
    mov $3, %ebx
    # IR: mov ebx, 3
    add %ebx, %eax
    # IR: add eax, ebx
```

| `PUSH src` | `push src` | 压栈 |
| `POP dest` | `pop dest` | 出栈 |
| `LEA dest, addr` | `lea dest, addr` | 加载有效地址 |

### 算术运算指令

| IR 操作 | x86 汇编 (Intel) | 说明 |

| `IMUL dest, src` | `imul dest, src` | 有符号乘法 |
| `IDIV src` | `idiv src` | 有符号除法 |
| `INC dest` | `inc dest` | 自增 1 |
| `DEC dest` | `dec dest` | 自减 1 |
| `NEG dest` | `neg dest` | 取反 |

### 逻辑运算指令

| IR 操作 | x86 汇编 (Intel) | 说明 |

## 🔍 高级用法

### 1. 使用寄存器映射

当编译器进行了寄存器分配后，可以使用寄存器映射：

```cpp
// 假设寄存器分配结果：vreg0 -> eax, vreg1 -> ebx
std::unordered_map<std::string, std::string> regMap;
regMap["vreg0"] = "eax";
regMap["vreg1"] = "ebx";

std::vector<IRNode> irNodes;
irNodes.push_back(IRNode(IROp::MOV, {"vreg0", "5"}, ""));
irNodes.push_back(IRNode(IROp::MOV, {"vreg1", "3"}, ""));
irNodes.push_back(IRNode(IROp::ADD, {"vreg0", "vreg1"}, ""));

IRToAssembly assembler(irNodes, regMap);
std::string assembly = assembler.compile();
```

**输出：**
```asm
    mov eax, 5
    mov ebx, 3
    add eax, ebx
```

### 2. 处理内存操作数

```cpp
std::vector<IRNode> irNodes;
irNodes.push_back(IRNode(IROp::MOV, {"eax", "[var_x]"}, "从内存加载"));
irNodes.push_back(IRNode(IROp::ADD, {"eax", "5"}, ""));
irNodes.push_back(IRNode(IROp::MOV, {"[var_x]", "eax"}, "存储回内存"));

IRToAssembly assembler(irNodes);
std::string assembly = assembler.compile();
```

**输出：**
```asm
    mov eax, QWORD [var_x]
    add eax, 5
    mov QWORD [var_x], eax
```

### 3. 生成完整的汇编程序

```cpp
std::stringstream fullProgram;

// 添加段声明和数据
fullProgram << "section .data\n";
fullProgram << "    var_x dq 0\n\n";
fullProgram << "section .text\n";
fullProgram << "global main\n\n";

// 生成代码
fullProgram << assembler.compile();

// 添加退出代码
fullProgram << "\n    mov eax, 0\n";
fullProgram << "    ret\n";

std::cout << fullProgram.str() << std::endl;
```

## 🎯 完整示例：计算阶乘

### IR 代码

```cpp
std::vector<IRNode> factorial_ir;

// int n = 5;
factorial_ir.push_back(IRNode(IROp::MOV, {"[n]", "5"}, ""));

// int result = 1;
factorial_ir.push_back(IRNode(IROp::MOV, {"[result]", "1"}, ""));

// for (int i = 1; i <= n; i++)
factorial_ir.push_back(IRNode(IROp::MOV, {"[i]", "1"}, ""));
factorial_ir.push_back(IRNode(IROp::LABEL, {".loop_start"}, ""));
factorial_ir.push_back(IRNode(IROp::MOV, {"eax", "[i]"}, ""));
factorial_ir.push_back(IRNode(IROp::CMP, {"eax", "[n]"}, ""));
factorial_ir.push_back(IRNode(IROp::JG, {".loop_end"}, ""));

// result = result * i
factorial_ir.push_back(IRNode(IROp::MOV, {"eax", "[result]"}, ""));
factorial_ir.push_back(IRNode(IROp::IMUL, {"eax", "[i]"}, ""));
factorial_ir.push_back(IRNode(IROp::MOV, {"[result]", "eax"}, ""));

// i++
factorial_ir.push_back(IRNode(IROp::MOV, {"eax", "[i]"}, ""));
factorial_ir.push_back(IRNode(IROp::ADD, {"eax", "1"}, ""));
factorial_ir.push_back(IRNode(IROp::MOV, {"[i]", "eax"}, ""));
factorial_ir.push_back(IRNode(IROp::JMP, {".loop_start"}, ""));
factorial_ir.push_back(IRNode(IROp::LABEL, {".loop_end"}, ""));

// return result
factorial_ir.push_back(IRNode(IROp::MOV, {"eax", "[result]"}, ""));
factorial_ir.push_back(IRNode(IROp::RET, {}, ""));
```

### 生成的汇编代码

```asm
section .text
global main

    mov QWORD [n], 5
    mov QWORD [result], 1
    mov QWORD [i], 1
.loop_start:
    mov eax, QWORD [i]
    cmp eax, QWORD [n]
    jg .loop_end
    mov eax, QWORD [result]
    imul eax, QWORD [i]
    mov QWORD [result], eax
    mov eax, QWORD [i]
    add eax, 1
    mov QWORD [i], eax
    jmp .loop_start
.loop_end:
    mov eax, QWORD [result]
    ret
```
